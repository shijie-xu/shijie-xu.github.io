Vmess+CDN+VPS=VPN never blocked

Jun 3, 2019

前端时间发现搬瓦工的VPN不能用了，经过检查，IP在国内被封。之前交了一年的VPS费用，却没有想到会出现这样的问题，于是找了一下解决方法。有一个思路就是利用CDN(Content Delivery Network)来解决IP blocked的问题。

原本的直连VPN网络拓扑：

your PC <—|Firewall|—> DNS host <——>VPS host <—|Firewall|—> target host

添加了CDN加速后的网络拓扑：

your PC <—|FW|—> CDN host <——> DNS host <——> VPS host <—|FW|—> target host

这样做的好处是，利用CDN服务器作为跳板，绕过了GFW，在CDN服务器几乎不可能被封的假设下，我们的VPN链路也是不可能中断的。

准备：需要一个域名，和CDN服务，互联网上有免费的服务可以使用。参考这篇文章
安装：参考这篇文章
使用：Client端使用的是v2rayN，需要仔细配置参数。
